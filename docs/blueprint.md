# **App Name**: Pasantías Manager

## Core Features:

- Registrar Pasantía Button: A button that takes the user to the 'Registrar pasantía' page.
- Ver Pasantías Button: A button that takes the user to the 'Ver pasantías' page.
- Centralized Layout: Main layout to render the buttons in a user friendly manner.

## Style Guidelines:

- Primary color: Blue (#007BFF) for a professional and trustworthy feel.
- Secondary color: Light gray (#F8F9FA) for backgrounds to ensure readability.
- Accent: Green (#28A745) for success states and primary actions.
- Clean and readable sans-serif font.
- Simple and intuitive layout with buttons centered on the main page.
- Subtle button hover effects for better user feedback.
